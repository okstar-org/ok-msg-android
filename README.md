# 🎁 项目介绍

**OkMSG 面向企业内部沟通的开源即时通讯软件，可以实现两人或多人之间的实时沟通！**
- 交流内容包括文字、语音、视频及文件等。
- 注重实用性、稳定性和扩展性，同时可以按照客户需求进行定制。
- 具备很好的跨平台性，支持Windows、Linux、macOS、Android、iOS等平台。
- 最大限度保障企业数据安全和隐私，支持消息端到端消息加密，支持企业独立私有化部署。


# 🧭 功能介绍

- 聊天模块
  - 具备单聊、群聊；
  - 支持文字、音视频、文件传输等基本功能；
  - 消息加密（计划中）。
